## Project Overview
This project is an interactive Power BI dashboard designed to analyze Dubai housing market trends and provide insights for investors, real estate agents, and policymakers.

## Objectives
- Analyze property prices by neighborhood
- Identify the most common property types and bedroom counts
- Compare Budget, Mid-Range, and High-End listings
- Understand how property age impacts price

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV

## Key Metrics
- Total Listings
- Average Price
- Average Size
- Highest Priced Property
- Average Price per Sqft

## Dashboard Features
- KPI Cards
- Interactive Slicers
- Map Visual
- Bar, Line, Treemap, and Stacked Column Charts
- Dynamic Insights Panel

## Key Insights
- Urban properties have the highest average price.
- Mid-Range properties dominate the market.
- Most listings are 2–3 bedroom homes.
- High-End properties generate the highest price per sqft.
